
# Employee Payroll App


## Use Cases (Short Summary)

### **1. Employee Registration**

**Concepts:** Encapsulation, constructor overloading, RegEx validation, composition  
**Features:** Validate email/phone/ID, create employee object, encrypt credentials

### **2. Authentication & Login**

**Concepts:** Inheritance, polymorphism, abstract classes, password hashing  
**Features:** Secure login, role-based access, session control

### **3. Payslip Generation**

**Concepts:** Composition, aggregation, Stream API, fluent design  
**Features:** Salary calculation, deductions, formatted payslip output

### **4. Payslip Download**

**Concepts:** equals(), hashCode(), cloning, immutability, file I/O  
**Features:** Generate downloadable PDF/text copy with unique filename

### **5. Dashboard Display**

**Concepts:** Interfaces, runtime type checking, Stream API, comparators  
**Features:** Recent payslips, YTD earnings, dynamic dashboard types

### **6. Input Validation**

**Concepts:** Custom exceptions, RegEx, sanitization  
**Features:** Centralized validation for all fields
